## ✨ Snow-Sakura主题特色

### 樱花主题配色
- 全新设计的 Sakura 配色方案（浅色/深色模式）
- 优化中文显示效果（霞鹜文楷字体）
- 专属链接颜色系统（内部/外部/悬停）

### 动态视觉效果
- **雪樱背景**：自适应的全屏背景
 
- **毛玻璃系统**：
  - 代码块磨砂质感
  - 文件列表/大纲适配深、浅色模式的毛玻璃效果
- 全界面透明化设计（完美展示背景）

### 功能增强
- 路径栏伪元素修复
- 文件列表选中状态优化
- 中文字体渲染优化
> [!IMPORTANT]
> 请安装Style Settings插件以启用 Sakura 配色和背景。
